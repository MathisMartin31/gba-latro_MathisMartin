Decimal Point Fonts

A set of macros to map the fonts decimal-point values (e.g. ".1") to their replaced characters.

"FP#" -> Font "Point" "number"

For example:

tte_printf("Testing " FP0_STR " Something!");

prints "Testing .0 Something!"

The map is the following:

'&' == '.0'
'^' == '.1'
'}' == '.2'
'{' == '.3'
'|' == '.4'
'`' == '.5'
'<' == '.6'
'>' == '.7'
'_' == '.8'
';' == '.9'
